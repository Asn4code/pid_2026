# Técnicas Computacionales en Biología

Materiales de la asignatura para el primer semestre del curso 2026-2027.

## Organización prevista

- `capitulos/`: temario redactado y versiones PDF.
- `presentaciones/`: diapositivas empleadas en clase.
- `practicas/`: actividades individuales, código y datos.
- `entorno/`: instrucciones y archivos para ejecución local o en la nube.
- `recursos/`: bibliografía y enlaces de consulta.

Las entregas y evidencias se registrarán exclusivamente en el Campus Virtual.
