# Datos de TC01-S01

`mini.fasta` contiene tres secuencias sintéticas creadas por Álvaro Serrano
Navarro para esta práctica. Se dedica al dominio público mediante CC0-1.0.

Su finalidad es practicar navegación, copia, inspección e integridad de
ficheros. No representa secuencias biológicas reales y no debe utilizarse para
extraer conclusiones biológicas.

La suma SHA-256 canónica se encuentra en `SHA256SUMS`. Para comprobarla:

```bash
cd datos
sha256sum -c SHA256SUMS
```
