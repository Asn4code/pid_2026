# Materiales docentes 2026-2027

Repositorio de apoyo para las asignaturas **Biología Computacional** y
**Técnicas Computacionales en Biología**.

## Asignaturas

- [Técnicas Computacionales en Biología](docs/tecnicas-computacionales-biologia/index.md)
- [Biología Computacional](docs/biologia-computacional/index.md)

## Uso del repositorio

Aquí se publicarán los capítulos, materiales de clase, prácticas, scripts,
entornos reproducibles y conjuntos de datos que puedan redistribuirse. Cada
práctica indicará cómo ejecutarla localmente y ofrecerá una alternativa en la
nube cuando sea necesaria.

El Campus Virtual es el canal oficial para avisos, entregas, evidencias,
calificaciones y retroalimentación. No deben publicarse entregas ni datos
personales del alumnado en este repositorio.

Los libros y demás recursos de terceros se citarán en cada módulo y se
consultarán mediante los servicios autorizados de la biblioteca o sus canales
oficiales. Su mención bibliográfica no implica que sus archivos puedan
redistribuirse aquí.

## Estado

La estructura y los contenidos corresponden al curso 2026-2027 y se irán
publicando por módulos.

El portal navegable se publica mediante GitHub Pages desde la rama `main`.
