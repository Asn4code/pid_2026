# Entorno de trabajo

Las prácticas se diseñan para ejecutarse primero en el equipo de cada
estudiante. Cuando el equipo o el sistema operativo no permita completar la
actividad, el mismo módulo ofrecerá un cuaderno equivalente en la nube.

## Ejecución local

Cada práctica incluirá:

- versiones de Python y demás herramientas;
- archivo de entorno o dependencias fijadas;
- instrucciones de instalación y comprobación;
- datos pequeños o procedimiento para obtenerlos;
- comando o cuaderno de inicio;
- salida mínima esperada.

## Alternativa en la nube

Cuando proceda, aparecerá un botón **Abrir en Colab** en la página del módulo.
El cuaderno utilizará los mismos datos, criterios y evidencias que la ejecución
local. Cambiar de entorno no modifica la actividad evaluable.

## Entregas

GitHub se utiliza para consultar y descargar materiales. No se aceptan
entregas mediante *issues*, *pull requests* o repositorios personales. Toda la
evidencia se entrega en la actividad correspondiente del Campus Virtual.

!!! warning "Datos y credenciales"
    No incluyas nombres, correos, claves, tokens ni datos sensibles en scripts,
    cuadernos, capturas o archivos compartidos.
